FAQ

How do I create an FAQ drop-down?

First, you will write the question and the answer. These should be on separate <p> lines. Select the question line by selecting the <p> node, then go to the Insert tab and select Drop-Down Text. This will create the FAQ drop-down and automatically place the question line within the <MadCap:dropDownHead> tags.

Next, place the answer line text in the <MadCap:dropDownBody> line that reads (This is the Drop-down text).

To apply the final styling, click your I-beam cursor into the question line (i.e. the MadCap:dropDownHead). Go to the Home tab > Style drop-down > then select the MadCap:dropDownHead.dropdownFAQ style.
