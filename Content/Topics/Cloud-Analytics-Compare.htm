Solstice Cloud Analytics: Compare

The Category Compare page allows you to compare Pod usage data based on the categories and options you select. Example.

This page is divided into two main parts:

  • Category Compare Graph - This is a graph at the top half of the page that displays data by category on a timeline.
  • Displays Table - This table can list all the Pods that are being managed in your Solstice Cloud instance, or it can display a filtered subset of Pods. Any filters you apply will affect the data that is displayed in the table. Clicking the check-boxes to the left of filtered Pods will show only the data from the selected Pods on the graph.

Collected Data

Solstice data is uploaded to Solstice Cloud on a daily basis. After it is uploaded to Solstice Cloud, the data is deleted from the Solstice Pod. Solstice Cloud tracks the following data for Solstice Pods.

Usage Data Room Occupancy Data

If your Pod is connected to a USB camera and has Room Occupancy enabled (Features settings), the following room occupancy data can be collected. As a note, room occupancy data is only tracked as an aggregate numerical value. No image or video ever leaves the Solstice Pod.

  • Average Hourly Occupancy: The average number of people detected in the room each hour. Hour increments are determined at the top of the hour. For example, 1:00 p.m. to 2:00 p.m. If a meeting runs from 1:45 p.m. to 2:15 p.m., the number of people in the room will be averaged in to both the 1:00 p.m. and 2:00 p.m. calculations.

  • Peak occupancy: The maximum number of people that have been detected in the room during the course of the meeting.

Insights from Analytics

Here are examples of how to generate graphs that can provide insights into how your Pods are being used.

How often are your meeting spaces used?

To determine how often meeting spaces that contain Mersive Pods are being used:

This example assumes you have defined a category named Room Type and that you have assigned Pods to options in that category. The category name used below is arbitrary, and if you have a category that is roughly similar, choose the name of that category instead in step 4.

  1. In the left sidebar menu of the Solstice Cloud portal, go to Analytics > Compare.

  2. At the top left of the page, click the time increment you want for your graph: Days or Weeks.

  3. At the top right of the page, click the time period you want the graph: 48 hrs, 7 days, 30 days, or Custom.

    For information on specifying a custom time frame, see Set a Custom Time Period for Your Graph .

  4. From the category drop-down, which is on the left just below the graph, choose Room Type.

  5. From the compare drop-down, choose Meetings.

  6. Using the check-boxes above the graph, select or unselect the options you want to display.

Each meeting space is rendered with a different colored line. By mousing over a plot point on the graph, you can display information about all the meetings spaces for that specific time period. Example

By selecting additional analytic data or categories in the Displays table, you can gain additional insights into the meeting context for each meeting. Example

How To

Below are the instructions on how to use the controls and filters on the Category Compare page.

Specify Time Increments for Your Graph Set a Custom Time Period for Your Graph
Select Category Options and Data to Compare
  1. Below the graph, click the left drop-down and select the category to graph.
  2. Click the right drop-down to select the data you want to graph. This will be the Y-axis of the graph.

    Above the graph, check-boxes appear that allow you to select the category options you want to graph.

  3. Check, or un-check, the category option boxes you want to graph.

Each category option check box that you select is graphed in a separate color.

 Filter the Pods in Your Data Set

By default, data from all the Pods being managed by Solstice Cloud is included in the graphs.

Search for Pods Manage Columns in the Displays Table

  1. In the upper-right hand corner of the Displays table, click the Manage Columns drop down.

    Check-boxes for all the data options and defined categories in your Solstice Cloud account appear. By default, any data or categories that you have selected to graph will be selected.

  2. Click the check-boxes for the data or categories you want to display as columns.

The selected data or categories appears in the Displays table. You can use the sort controls to the right of column labels to manipulate the sort order of the table.
