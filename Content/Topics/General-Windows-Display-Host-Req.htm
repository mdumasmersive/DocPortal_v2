Solstice Windows Display Software

Solstice Display Software is a version of Solstice that is installed on a Windows 7, 8, or 10 host PC and connected to the display by HDMI video cable. Note: Solstice Display Software for Windows is not supported in a virtual machine (VM) environment or while using terminal services.

Hardware Requirements for Host PC

  Minimum Recommended
Processor Intel i5 or AMD Ryzen 5 or higher Intel i7 or AMD Ryzen 7 or higher, 4+ core recommended
RAM 4 GB 8 GB or higher
Max Display Resolution 1080p 1080p or 4K

 
Operating System

Windows 7, 8, or 10

Note: Solstice Windows Display Software is not supported in a virtual machine (VM) environment or while using terminal services.

Graphics

DirectX 11 or higher, or Dedicated Graphics (i.e. AMD Radeon or NVIDIA GeForce)

 

Network

Single NIC Card, Ethernet connection

 
If adding an external video and audio source input

These DirectShow compatible frame grabber cards are supported.

If audio support needed:

  • Avermedia DarkCrystal HD Capture SDK II (C729) - (1 HDMI, 1 analog input)
  • VPrime XI200DE-HDMI - (2 HDMI inputs)

If protected content support needed (no audio support):

  • Datapath VisionRGB-E1S - (1 DVI-I input)
  • Datapath VisionRGB-E2S - (2 DVI-I inputs)
