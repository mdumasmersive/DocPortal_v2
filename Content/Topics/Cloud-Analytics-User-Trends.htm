How To

Below are the instructions on how to use the controls and filters on the User Trends page.

Set a Custom Time Period for Your Charts and Graphs
Filter the Pods in Your Data Set by Category

By default, data from all the Pods being managed by Solstice Cloud is included in the charts and graphs. You can include or exclude the data from a specific Pod or Pods using the Filter option.

  1. In the upper-left hand corner of the page click .

    The Filter by Category screen displays on the right.

  2. Click the drop-down controls for each category to reveal the available category options.

    Clicking on any of the categories displays all the defined options for that category and indicates how many displays have that option set. When you select one or more options under a single category, the table will display all Pods that belong to ANY of the selected options. However, if you make selections in a second category, the table will only display Pods that belong to any option selected in the first category AND any option selected in the second category.

  3. Click the check-boxes you want to filter with.
