Solstice Cloud Analytics: In-Room Tech

The In-Room Tech Analytics page allows you see how teams collaborate when they come together in meeting and learning spaces by providing data on the types of devices that are connecting to your organization's Solstice Pods, and the methods that are being used to share content. Additionally, for Pods that are being used to support web conferencing, you can display data about the number of meetings that use conferencing (i.e. "hybrid meetings) as well as the web conferencing platforms that are shared to Solstice (e.g. Zoom, Teams, etc.). How To

Below are the instructions on how to use the controls and filters on the In-Room Tech page.

Search for Pods
  1. In the upper-left hand corner of the In-Room Tech table, enter text in the Search field.

A filter is set and the table shows the Pods that match your text.

The search filter matches text that may not be displayed on the page. For example, if the text you enter matches a option specified in a category, then that Pod will be included in the list. If it is not apparent why Pods are matching your search text, use the Filter feature on the In-Room Tech Analytics page to look at the Filter by Category options that may be matching your query.

 Filter the Pods in Your Data Set by Category

By default, data from all the Pods being managed by Solstice Cloud is included in the charts and graphs. You can include or exclude the data from a specific Pod or Pods by manually checking or unchecking the box to the left of the Pod name in the Displays table.

Only checked Pods are included in the data that is graphed.

You can also check or uncheck the "select all" box to the right of the Name column header to select or clear all checked Pods
