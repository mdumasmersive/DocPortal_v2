Deploy the Solstice App with MSI or SCCM

For ease in centrally deploying the Solstice App, Mersive provides both an MSI and an SCCM installer package of the Solstice App for Windows. The MSI installer package allows for a GUI-based installation on a local machine or GPO deployment in Active Directory, while the SCCM installer package allows for a remote installation. Either of these methods can be used to deploy the Solstice App silently by your IT administrator.
