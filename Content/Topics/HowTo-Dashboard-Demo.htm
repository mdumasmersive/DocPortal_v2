How to Install and Run the Dashboard Demo

  1. Visit mersive.com/download and click on Management Tools.
  2. Under Solstice Dashboard, click the Download Solstice Dashboard link.
  3. Fill out the download form then click Submit.
  4. Run the SolsticeDashboardSetup.exe installer and step through the InstallShield wizard.
  5. On the Select Features portion of the installation process, enable the Demo option.

  6. When the installation process is completed, run the Start Solstice Demo Hosts program.
  7. If prompted, opt for the Windows Firewall to allow access for the Solstice display simulator program. A Solstice command prompt window will appear.
  8. Enter the number of Solstice demo host instances you would like to launch (5 instances is recommended) and hit your Enter key.
  9. Enter a name for your Solstice demo host instances and hit Enter. If no name is entered, the default name “DemoHost” will be applied, and each instance will be assigned a number starting with “01”.

  10. Leave the Solstice command prompt window open, and launch the Dashboard application. Once launched, go to the Import Displays section and click the Discover button. A pop-up window appears.
  11. Find the Solstice demo instances you created, shift-click to select them, then click Import.

    12. Your Solstice demo hosts will appear in the instances panel (left side) of your Solstice Dashboard. From here, you can select one or more Solstice demo host instances to virtually manage by updating the various options and settings that appear on the right side of the Solstice Dashboard interface. For a full overview of the functional capabilities of the Solstice Dashboard, see Get Started with Solstice Dashboard.
